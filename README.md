## Setup

### Prerequisites

Android Studio  
Java  
Git  

### Setup Steps

Clone this repository locally using 
  ```sh
  git clone https://github.com/audreytracy/android-pathfinding.git
  ```
In Android Studio, go to File -> Open to find & open the project.
